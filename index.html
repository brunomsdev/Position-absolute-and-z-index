When you apply position: absolute, the elements are going to:
1. Be dependent on the webpage (body) or the parent container (if this one has position: relative applied);
2. Be in different layers which means that the images will be on top of each other in the order that it is organized in the HTML file, being the first image behind the second, and the second behind the third and the third behind the fourth.
3. If the position isn't declared, the images will be on top of each other on the top left corner.

Position: absolute has a huge advantage placing the specific position of an image or element on the webpage or container. The properties top, right, bottom and left are used for placing an element. These images have the following coordinates:
Hat - top: 20px; left: 650px;
Astronaut helmet - top: 20px; right: 100px;
Bag - bottom: 20px; left: 650px;
Cassette tape - bottom: 20px; right: 100px;

Position: absolute makes the elements be organized by layers being the fisrt element in the HTML file on the back followed by the second, third and so on. The last element/image is in the front of the others.

It is possible to change which image is in front of the other by changing the order of the elements in HTML file.

Another way to chage the order of the layers is to use the property z-index. Z-index can have a value from 0 to 99. It means that a higher value will be in front of the page and a lower in the back of it.

#hat {
top: 150px;
left: 400px;
}
#astronaut-helmet {
top: 170px;
left: 500px;
z-index: 1;
}
#bag {
top: 190px;
left: 600px;
}
#cassette-tape {
top: 210px;
left: 750px;
}

The astronaut helmet is in the exactly same position as the previous page. What makes it to go to the front is the z-index:1;
